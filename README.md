# Wall Calendar

An interactive wall calendar component built with Next.js + Tailwind CSS.

## Features
- Physical wall calendar aesthetic with hero image
- Date range selection with visual highlighting
- Notes per date/range (persisted in localStorage)
- Fully responsive (desktop side-by-side, mobile stacked)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Tech Stack
- Next.js 14 (App Router)
- Tailwind CSS
- date-fns
- TypeScript
